Portfolio Website

November 15, 2023 · Cole Sandlin

I built a static website using the Hugo framework to act as a personal portfolio. I chose Hugo because it’s open-source and has a fast generation speed. I selected the PaperMod theme to use as a template due to its minimalist appearance and personalized it with custom HTML. For hosting, I opted to use GitHub Pages because it’s free and I was already familiar with Git.
